# TruSaba Brand Spec

Friendly AI travel companion for young travellers — trustworthy, rounded, card-based, never corporate.

## Tokens (OKLch)

```css
:root {
  --bg: oklch(1.000 0.000 89.9);
  --surface: oklch(0.984 0.003 247.9);
  --fg: oklch(0.218 0.000 89.9);
  --muted: oklch(0.551 0.023 264.4);
  --border: oklch(0.928 0.006 264.5);
  --accent: oklch(0.552 0.184 255.7); /* #066FDA */
  --secondary: oklch(0.847 0.170 87.3); /* #FCC415 */
  --success: oklch(0.627 0.170 149.2);
  --warn: oklch(0.795 0.162 86.0);
  --danger: oklch(0.577 0.215 27.3);
}
```

## Type

- Display / body: `"Plus Jakarta Sans", system-ui, sans-serif`
- Mono (codes, prices): `ui-monospace, "SF Mono", Menlo, monospace`

## Posture

1. Mobile-first phone shell ~390px; generous whitespace, 14–16px card radius, soft elevation.
2. Primary blue only for main CTAs and key interactive states; gold for badges/highlights (max 1–2 gold per screen).
3. Card-first layouts; monoline SVG icons; travel-companion microcopy (warm, short Indonesian).
4. SOS always danger red; status greens for success states.
5. Logo (`logo.jpeg`) as brand mark in headers/onboarding — infinity trail + plane.
